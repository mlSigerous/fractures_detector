# Fracture Detector

Ссылка на видеопрезентацию проекта: [ссылка](https://youtu.be/YZ6cxHjAtP4)

Учебный проект для локализации возможных переломов на рентгеновских снимках.
Юзер загружает изображение в Streamlit-приложение, после чего модель
YOLO отмечает подозрительные области bounding box`ами и показывает
уверенность предсказаний.

В приложении используется checkpoint detection-модели **YOLO26m** с лучшим
полученным результатом:

- **test mAP@0.5: 0.2058**;
- задача сведена к одному классу `fracture`;
- размер входного изображения — 768 px.

`train_model.ipynb` содержит более поздний эксперимент с instance segmentation
на очищенных от пересечений сплитах. Модель YOLO26l-seg достигла
`validation box mAP@0.5 = 0.2401`, но показала только
`test box mAP@0.5 = 0.1562`, поэтому в приложение она не включена.
